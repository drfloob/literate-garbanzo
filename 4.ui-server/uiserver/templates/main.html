About Network Pulse

Hi, I'm AJ.

I created Network Pulse over 3 weeks as part of the Insight Data Engineering Fellowship, Silicon Valley, Fall 2016.

Network Pulse is a distributed, fault-tolerant big data pipeline that performs graph analysis over unbounded data streams. Its purpose is to find clusters of people who have interacted with each other -- directly or indirectly -- in a short time frame, and display those results are they're happening.

For more background on the project, the slides are available online, and the project is Open Source.
